# Beszel Hub Add-on Documentation

## Installation

Add this repository to Home Assistant and install the Beszel Hub add-on.

## First Time Setup

1. Start the add-on
2. Click "OPEN WEB UI"
3. Create your admin account
4. Configure your monitoring preferences
